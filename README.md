## 4.3数据库设计
数据库是信息系统的基础和核心。数据库设计的好坏直接影响到信息系统开发的成败。创建数据库表首先确定实体的属性和实体之间的关系。根据关系创建一个数据表。
### 4.3.1实体ER图
数据库是整个软件编程中最重要的一个步骤，对于数据库问题主要是判定数据库的数量和结构公式的创建。外来人员管理系统使用的是Mysql进行对数据库进行管理，进行保证数据的安全性、稳定性等。
